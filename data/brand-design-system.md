# Shamy — Brand Design System

> Generated using the *ui-ux-pro-max* methodology (Reasoning Engine v2.0), adapted for a pet/lifestyle social-content brand.
> This file is the **single source of truth** for visual identity decisions. Anything that touches Shamy visually (IG profile, X header, TikTok cover, Stories templates, post overlays, watermarks) must read from here.

---

## 1. Brand-input recap

- **Product type:** Lifestyle / pet-content social brand.
- **Audience:** Cat lovers, rescue advocates, general "wholesome internet" viewers.
- **Mood:** Warm. Cozy. Slightly mischievous. Gentle. Real (not over-polished).
- **Origin:** Calw / Stuttgart, Germany — natural light, German window frames, leafy outdoor shots.
- **Subjects:** Amy (fluffy brown tabby Maine Coon style) + Sheldon (spotted "cheetah" Bengal style).
- **Anti-personality:** Glossy, aspirational, over-curated, AI-poster aesthetic.

## 2. Recommended pattern

**Pattern:** `Brand Storytelling + Social Proof`
Conversion logic: emotion-driven; the brand is the story (rescue glow-up); proof is the cats themselves.

**Page-section recipe (Linktree / future website if any):**
1. Hero portrait (both cats together)
2. The brand story (rescue arc, short)
3. Latest 6 posts grid
4. Where to follow (IG / X / TikTok buttons)

## 3. Recommended UI style

**Primary style:** `Organic Biophilic` × `Soft UI Evolution` hybrid.
- *Organic Biophilic* = wellness/sustainability brand DNA — natural shapes, leafy textures, calming.
- *Soft UI Evolution* = soft shadows, subtle depth, premium-but-friendly.

**Secondary touchpoint (for memes only):** `Neubrutalism-lite` — chunky text, hand-feel stickers, used sparingly for snarky cat-POV captions.

## 4. Color palette

Anchored on the cats' own coat tones so the brand visually rhymes with the subjects.

| Role | Hex | Name | Where it shows up |
|---|---|---|---|
| Primary | `#C8A27A` | Amy Cream | Profile accent, Story background blocks |
| Secondary | `#7A5A3F` | Sheldon Brown | Text overlays, emphasis |
| Tertiary | `#A8B89A` | Window-Sage | Background panels, IG Highlight covers |
| Soft background | `#FAF5EE` | Warm Linen | Story backgrounds, watermark plate |
| Accent / CTA | `#E07A5F` | Dusty Terracotta | "Follow" buttons, link stickers |
| Text-on-light | `#2E2A26` | Charcoal Cocoa | Body text |

**Anti-palette (never use):** neon pinks, electric blues, "AI purple/pink" gradients, pure white `#FFFFFF`, pure black `#000000`. They fight with the natural photography.

## 6. Visual effects & motion

- Shadow: soft, low-spread, never harsh — `0 8px 24px rgba(46, 42, 38, 0.08)`.
- Corners: 16–24 px on plates and Story templates.
- Animation: 200–300 ms ease-in-out for any motion. Respect `prefers-reduced-motion`.
- Photo treatment: keep originals. **Do not over-filter.** Natural light is the look.
- Transitions in slideshow videos: simple cross-fade (300 ms), Ken Burns on every still by default.

## 7. Iconography & emoji

- **Icons:** Lucide outline set (1.5 px stroke) when icons are needed for Story stickers, web buttons.
- **Emoji set in copy:** 🐾 🍓 🪴 🌿 ☁️ 🎀 🥖 🌞 ✨ 🐈 🐈‍⬛
- **Banned:** Don't use emoji as functional icons (use Lucide SVG instead). Don't pile on more than 2 emoji per caption.

## 9. Pre-delivery checklist (must pass before every post)

- [ ] No Sneha or Anish visible (privacy check passed).
- [ ] Image contrast 4.5:1 if any text overlay.
- [ ] Caption length within platform recommendation (IG: <125 chars before "more"; X: <240; TikTok: <100 over the photo).
- [ ] Hashtag count 5–8; mix of branded + niche.
- [ ] Trending sound link present (TikTok only).
- [ ] No emoji used as icons.
- [ ] No "AI gradient" filters.
- [ ] Cat names spelled correctly (Amy & Sheldon, capitalised).

## 10. Anti-patterns (industry rules for pet content)

| Avoid | Why |
|---|---|
| Heavy filters / over-saturation | Reads as "stock photo," kills authenticity |
| Generic stock-photo overlays | Audience smells inauthenticity |
| Posting cats in distress for engagement | Ethical, also depresses retention |
| Using human family member's voice/face | Violates the privacy boundary |
| Begging for follows in captions | Algorithm hates it, audience hates it |
| Daily "follow for more cat content" CTAs | Use sparingly, max once per week per platform |
